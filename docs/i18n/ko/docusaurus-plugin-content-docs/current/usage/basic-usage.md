# 기본 사용법

Security Scanner MCP로 코드와 인프라를 스캔하는 방법을 배우세요.

## 코드 스캔

### 인라인 코드

```
이 코드를 스캔해줘:

const password = "admin123";
```

### 파일 스캔

```
src/auth.js를 스캔해줘
```

### 여러 파일

```
src/ 디렉토리의 모든 JavaScript 파일을 스캔해줘
```

## IaC 스캔

### Dockerfile

```
Dockerfile을 스캔해줘
```

### Kubernetes

```
deployment.yaml을 스캔해줘
```

### Terraform

```
main.tf를 스캔해줘
```

## 수정 받기

```
이전 취약점을 수정해줘
```

## 리포트 생성

```
전체 보안 리포트를 다이어그램과 함께 생성해줘
```
