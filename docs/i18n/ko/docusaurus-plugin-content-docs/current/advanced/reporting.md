# 리포트

종합적인 보안 리포트를 다양한 형식으로 생성합니다.

## 리포트 형식

### Markdown

```
Markdown 리포트를 생성해줘
```

### SARIF

GitHub Code Scanning과 호환:

```
SARIF 형식으로 내보내줘
```

### HTML

```
HTML 리포트를 생성해줘
```

## 리포트 내용

- 📊 요약 통계
- 🔍 상세 취약점
- 📈 Mermaid 다이어그램
- 💡 수정 권장사항
- 🏷️ OWASP/CWE 매핑

## 사용 예제

```
전체 보안 리포트를 다음과 함께 생성해줘:
- 심각도별 요약
- Mermaid 다이어그램
- 각 이슈에 대한 수정 제안
```
